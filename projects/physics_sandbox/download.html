Download and Installation Guide

Installation Controls Performance Future Enhancements

Installation

Prerequisites

  • Python 3.x installed on your system
  • Git (optional, for cloning the repository)

Download Options

Option 1: Direct Download

  1. Download the latest release from GitHub
  2. Extract the files to your desired location
  3. Open a terminal in the project directory
  4. Run: pip install -r requirements.txt

Option 2: Clone Repository

  1. Open a terminal
  2. Run: git clone https://github.com/cardsea/phisics-game.py.git
  3. Navigate to the project directory
  4. Run: pip install -r requirements.txt

Running the Game

After installation, run the game using:

python main.py

Controls

Mouse Controls

  • Left Click: Create/Place objects
  • Drag: Draw walls or create multiple objects
  • Right Click: Erase objects

Keyboard Controls

  • 1-4: Select different ball types
  • W: Select wall tool
  • E: Select eraser tool
  • Space: Pause/Resume simulation
  • R: Reset simulation

Performance Optimization

System Requirements

  • Minimum: 2GB RAM, any modern CPU
  • Recommended: 4GB RAM, 2GHz+ CPU
  • Graphics: Any modern GPU

Optimization Tips

  • Limit the number of objects for better performance
  • Use simpler wall shapes when possible
  • Close other resource-intensive applications
  • Update graphics drivers regularly
